B2B대금수납 연계 API

정보출처: 국내 21개 은행

#B2B매출내역 #전자어음수탁내역

보유한 대금수납내역 (B2B매출내역)을 조회하여 ERP로 연계합니다.

제공 API

B2B상품 매출내역
(타행전자결재통합조회)
B2B상품, 기관, 구매기업명, 채권발행일자, 채권만기일자, 은행고유채권번호, 입금금액, 입금통화내용, 품목명, 메모,
적요, 발행한도금액 등

정보제공기관

산업은행, 기업은행, 국민은행, 수협은행, 농협은행, 우리은행, SC은행, 씨티은행, 대구은행, 부산은행, 광주은행, 제주은행, 전북은행, 경남은행,새마을금고, 신협은행,
우체국, KEB 하나은행, 신한은행, K뱅크, 산림은행 등 (국내 21개은행)

연계효과

  • 거래은행의 모든 B2B매출내역을 ERP에서 바로 조회할 수 있습니다.

    여러 은행의 다양한 B2B상품을 일일이 조회하지 않고 ERP에서 바로 은행B2B매출대금결제 현황을 확인하고 관련 전표를 처리할 수 있습니다. 특히 다수 사업장을 보유한 경우 업무부담이 크게 줄어듭니다.

  • 은행 B2B상품매출내역 통합조회 및 연계로 ERP에서 현금흐름 관리가 편하고 정확해집니다.

    B2B상품매출내역의 판매금액, 만기일, 할인가능금액을 모두 제공하므로 ERP에서 가용자금 관리가 쉬워지고
    기존 수기 입력에 따른 누락, 오입력의 위험 없이 정확한 매출채권관리가 가능합니다.

연계사례

  • SAP ERP

    “전자어음 내역 연계로 수기 입력 최소화”

      • 고객명: S사
      • 업종: 정보통신업
      • 매출액: 500억 이상

    전자어음 등 B2B 매출내역을 각 은행 인터넷뱅킹에 접속하여 조회 후 데이터를 ERP에 수기로 입력하여 전표를 작성하였으나,
    B2B대금수납 연계를 통해 거래 데이터를 ERP로 간편하게 이관하여 불필요한 수기 입력 최소화

  • 영림원소프트랩

    “수기 입력을 자동화하여 오입력 및 누락건 발생 예방!”

      • 고객명: C사
      • 업종: 화공약품 제조
      • 매출액: 1,000억 이상

    B2B 매출내역을 수기 입력하면서 매출번호 오입력 건이 상당수 발생, 수정 작업에 2시간이 소요되었으나,
    B2B대금수납 연계 후 최신 B2B상품 거래내역을 ERP로 전송, 수정 및 삭제가 불가능하도록 설정하여 누락 건 발생 예방
