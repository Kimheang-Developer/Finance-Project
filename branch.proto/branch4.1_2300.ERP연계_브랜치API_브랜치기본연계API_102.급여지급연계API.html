연계효과

  • 급여시스템에서 확정한 지급내역을 바로 이체 할 수 있어 안전하고 편리합니다.

    복잡한 파일 등록, 변환 등의 수작업 없이 지급파일을 은행으로 전송할 수 있어 안전하고 빠르게 대금지급 업무를 처리할 수 있습니다.
    급여시스템에서 작성한 직원별 급여액을 공개하지 않고 지급할 수 있습니다.

  • 자금사고 예방을 위한 내부통제 강화를 기대할 수 있습니다.

    내부 시스템에서 지급승인이 완료된 데이터는 임의로 삭제 및 변경 할 수 없습니다.
    지급 실행 시 은행 기업인터넷뱅킹 수준의 다단계결재, 인증서 제출 등의 내부통제 과정을 적용하여 자금사고를 예방 할 수 있습니다.

연계사례

  • 영림원소프트랩

    “ERP연계를 통해 급여지급 프로세스를 구현하여 업무 효율성 강화”

      • 고객명: A재단
      • 업종: 비영리/협회

    내부 인사정보를 바탕으로 ERP시스템에서 만들어진 임직원 급여정보를 브랜치로 전송하여 지급함으로써,
    급여 지급 정보가 공개되지 않아 보안을 강화하고 업무 효율성을 향상

  • 더존비즈온

    “급여지급 연계를 통한 자금통제 기능 향상 및 지급업무 자동화”

      • 고객명: L공단
      • 업종: 공기업
      • 사원수: 1,000명 이상

    인재 개발부에서 작성된 임직원 급여정보를 총무 지급담당자가 위, 변조 없이 연계 지급함으로
    자금사고 예방 및 지급업무 자동화
