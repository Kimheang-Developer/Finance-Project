기업이 보유한 증권계좌, 보험계약 내용을 한번에 조회하여 손쉽게 관리합니다.

제공 API

정보제공기관

국내 모든 증권사, 보험사

연계효과

  • 여러 증권사에 흩어져있는 증권계좌를 한번에 조회 할 수 있습니다.

    증권사에서 제공하는 인터넷뱅킹에 일일이 접속하지 않아도 모든 증권사의 계좌현황을 한번에 조회 할 수 있습니다.
    기업이 보유하고 있는 계좌 현황을 한번에 파악하여 기업자금관리에 활용해 보세요.

  • 국내 모든 보험사의 B보험계약 내용을 조회하여 보험자산을 손쉽게 관리합니다.

    국내 모든 보험사의 계약내용을 상세하게 확인할 수 있습니다. 별도 SW설치 없이 간편하게 API호출로 서비스를 구현합니다.
    기업이 가입한 보험상품의 목록을 확인하시고 손쉽게 관리하세요!
