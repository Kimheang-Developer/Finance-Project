연계효과

  • 외화계좌입출금내역을 연동하여 전표처리를 쉽고 빠르고 정확하게 할 수 있습니다.

    ERP에서 계좌 입출금내역을 바로 확인하면서 전표처리를 하므로 쉽고 빠르고 정확한 업무가 가능합니다.

  • 환율정보 연계와 함께 사용하면 정확한 원화 환산 금액을 쉽게 관리할 수 있습니다.

    외화거래내역 연계 시 통화코드를 제공하므로 환율정보연계서비스와 함께 사용 시 정확한 원화환산금액 관리로
    회계처리를 정확하게 할 수 있습니다.

연계사례

  • 더존비즈온

    “외화무역 연계를 통한 외화계좌 입출내역 자동화”

      • 고객명: T사
      • 업종: 통신업
      • 매출액: 1,000억 이상

    외화무역 연계를 통해 ERP에 약속된 계정과목과 입금내역이 자동 매핑함으로써 일반계좌 및 외화계좌 입출내역 대사 업무를 자동화하고,
    이관 받은 데이터로 전표처리 업무를 간소화하여 업무시간 단축

  • 영림원소프트랩

    “입출금내역 전표자동화 및 오입력을 방지”

      • 고객명: M사
      • 업종: 반도체 제조
      • 매출액: 1,000억 이상

    외화계좌의 거래내역을 각 은행 인터넷뱅킹에서 스크래핑 후 브랜치에서 ERP로 전송함으로써
    입/출금 내역 전표 자동화 및 수기 입력 시 발생 할 수 있는 오입력을 방지, 쉽고 빠르고 정확한 업무 가능
