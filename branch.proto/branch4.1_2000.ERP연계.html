ERP에 금융을 연결하세요.
자금업무가 편하고 안전해집니다.

브랜치API로 ERP에 금융을 연결하세요.

  • 대금지급 연계 API

    ERP에서 확정한 거래처 등 대금지급내역을
    바로 이체 할 수 있어 자금사고를 예방합니다.

  • 급여지급 연계 API

    급여시스템에서 확정한 지급내역을
    바로 이체 할 수 있어 안전하고 편리합니다.

  • 원화 수시입출계좌 연계 API

    모든 거래 은행의 원화계좌 입출내역을 연계합니다.
    수금확인과 전표업무를 개선합니다.

  • 자금대체(집금)결과 연계 API

    집금결과를 연계하고 대체전표를 자동화합니다.
    자금집금으로 자금운용 효율이 개선됩니다.

Q 연계 이전의 어려움은?
기존에는 담당자가 아침마다 은행 뱅킹의 최초 고시환율을 확인한 후 외화계좌 보유 은행 인터넷뱅킹에 각각 로그인하여 거래내역을 확인하고 수기로 ERP시스템에 입력하느라 시간이 많이 소요되고 변화하는 환율 정보를 매번 입력하는 과정에서 잘못 입력하는 경우도 많았습니다.
Q. 연계 후 어떤 점이 좋아졌나요?
외화계좌의 거래내역을 자동으로 회계시스템으로 연계하여 회계전표 자료로 이용하고 통화별 환율을 자동으로 조회하여 ERP에 연계하므로 오·입력 없이 빠르고 정확하게 기준환율을 반영하게 되었음. 이로 인해 원화환산시재와 외환차손익 관리가 빠르고 정확해졌습니다.

외화계좌의 거래내역을 자동으로 회계시스템으로 연계하여 회계전표 자료로 이용하고 통화별 환율을 자동으로 조회하여 ERP에 연계하므로 오·입력 없이 빠르고 정확하게 기준환율을 반영하게 되었음. 이로 인해 원화환산시재와 외환차손익 관리가 빠르고 정확해졌습니다.
