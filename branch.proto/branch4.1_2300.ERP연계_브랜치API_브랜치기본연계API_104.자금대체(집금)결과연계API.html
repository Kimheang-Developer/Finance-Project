연계효과

  • 집금결과를 연계하여 대체전표를 자동화 합니다.

    Idle Money 최소화를 위해 정기적인 자금 집금을 할 경우 대체전표가 많아집니다. 집금 결과 ERP연계로 대체전표 처리를 자동화합니다.

연계사례

  • SAP ERP

    “모계좌로 한번에 집금하고 대체전표도 자동화”

      • 고객명: A업체
      • 업종: 도소매
      • 매출액: 5,000억 이상

    브랜치의 자금집금 기능을 통해 44개의 계좌를 모계좌로 한번에 집금함으로써 유휴자금 없이 효율적인 자금 운영이 가능하고,
    집금결과 내역은 입출금거래내역과 구분하여 ERP로 전송하여 대체전표 자동화

  • 영림원소프트랩

    “정확하고 빠르게 시스템에 반영하고 대체전표 일괄처리로 업무비용 절감”

      • 고객명: W제약
      • 업종: 의약품 제조
      • 매출액: 5,000억 이상

    자금대체(집금)결과 연계로 재무팀의 대체전표 작성 시 발생할 수 있는 누락과 오입력 없이 정확하고 빠르게 회계시스템에 반영하고
    대체전표 일괄처리로 업무비용 절감 효과
